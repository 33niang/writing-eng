## 📘 README

### 🧩 项目简介

本项目是一个用于 **考研英语作文学习与背诵** 的网站，主要功能包括中英文双语展示与点击翻译功能，方便用户在阅读与背诵时快速理解句子含义。

### ✍️ 内容来源

* **作文模板**：出自 **石雷鹏《考研英语作文冲刺背诵20篇》（2026版）**。
* **Essay 真题**：来源于 **历年考研英语（一、二）真题作文**。

所有素材仅用于学习与研究目的。

### 💡 功能特色

* 支持中英文对照展示；
* 点击英文句子即可显示中文翻译；
* 点击其他句子或页面空白区域可自动关闭当前翻译；
* 采用全局状态管理（`store.js`）确保同一时间仅显示一个翻译窗口；
* 结构基于 **VitePress** 自定义主题开发，组件为 `Bilingual.vue`；
* 可部署在 **Cloudflare Pages** 或 **EdgeOne 静态托管平台** 上，实现全球加速与自动构建。

### 🛠 技术栈

* **框架**：Vue 3 + VitePress
* **语言**：JavaScript / Markdown
* **样式**：CSS（Scoped）
* **状态管理**：轻量自定义 Store

### 📁 目录结构

```
docs/
 ├─ .vitepress/
 │   ├─ theme/
 │   │   ├─ components/
 │   │   │   └─ Bilingual.vue   # 翻译组件
 │   │   ├─ store.js            # 全局状态管理
 │   │   └─ index.js            # 注册全局事件
 │   └─ config.js
 └─ essays/
     ├─ templates/              # 作文模板（石雷鹏）
     └─ real/                   # 真题作文
```

### ⚙️ 部署建议

1. 构建命令：

   ```bash
   npm run docs:build
   ```
2. 构建输出目录为：

   ```
   docs/.vitepress/dist
   ```
3. 将该目录上传或绑定至

   * **Cloudflare Pages**（推荐）
   * 或 **EdgeOne**（支持自定义域名与 CDN 加速）
